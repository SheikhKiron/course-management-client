A modern Course Management System with Next.js frontend and Node.js/Express backend, powered by MongoDB. Manage courses, authentication, and enrollment with a responsive UI.

Features

Firebase Email/Password & Google Authentication

Add, update, delete, and view courses

Responsive design with Tailwind CSS & DaisyUI

Toast notifications for actions

Dynamic routing & client-side rendering

Node.js + Express backend with MongoDB for data storage

Tech Stack

Frontend: Next.js 16 | React 18 | Tailwind CSS | DaisyUI | React Toastify | Vercel
Backend: Node.js | Express | MongoDB

Setup
Frontend
git clone https://github.com/SheikhKiron/course-management-client.git
cd course-management-client
npm install
npm run dev


Add .env.local with your Firebase credentials.

Backend
git clone https://github.com/SheikhKiron/course-management-server.git
cd course-management-server
npm install
npm run dev
